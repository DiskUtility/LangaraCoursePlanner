Langara Course Planner

Thank you for using the Langara Course Planner!

I created this website after facing constant frustration with Langara's default course selection interface.

I felt that there must be an easier way to plan courses - so I created one.


This website uses information from the Langara website and BCTransferPlanner. Data for the last semester is retrieved every hour.

This website is powered by coursesapi.langaracs.ca.

You can search by subject, course code, course title, course description, location of course (e.g. T001), and instructor.
