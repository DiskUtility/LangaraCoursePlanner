Langara Course Planner

Thank you for using the Langara Course Planner!

I created this website after facing constant frustration with Langara's default course selection interface.

I felt that there must be an easier way to plan courses - so I created one.


This website uses information from the Langara website and BCTransferPlanner. Data for the last semester is retrieved every 30 minutes.

You can also use AND, OR, and NOT operators in the search bar. You can search by subject, course code, location, class type, or by teacher.
